# Amazon Homepage Clone

A responsive Amazon homepage clone built using HTML and CSS.

## Overview

This project recreates the user interface of Amazon's homepage to practice modern frontend development concepts including:

- Semantic HTML5 structure
- CSS Flexbox layouts
- Responsive design principles
- Navigation bar design
- Hero section implementation
- Product cards and product grids
- Horizontal product slider
- Multi-section footer

## Features

- Amazon-inspired navigation bar
- Hero banner section
- Product category cards
- Horizontal scrolling product slider
- Responsive layout using Flexbox
- Interactive links and hover effects
- Structured footer with multiple sections

## Technologies Used

- HTML5
- CSS3
- Flexbox
